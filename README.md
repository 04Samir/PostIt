## Installation
Clone the Repository:
```bash
$ git clone https://github.com/04Samir/PostIt.git
```

Navigate to the Directory:
```bash
$ cd PostIt
```

Install the Dependencies:
```bash
$ npm install
```


## Usage
Start the Server:
```bash
$ npm start
```

Now you Can Visit [`localhost:1234`](http://localhost:1234) To View The Application!
